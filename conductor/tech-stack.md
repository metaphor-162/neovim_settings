# Tech Stack

## Core Editor
- **Vim / Neovim**: The primary text editor environment, functioning locally on remote servers.

## Configuration Language
- **Vimscript (VimL)** / **Lua**: The language(s) used for defining editor settings, keymaps, and managing plugins.

## Automation & Scripting
- **Bash / POSIX sh**: Used for the automated, zero-dependency installation script, ensuring compatibility across various Unix-like systems.

## Version Control
- **Git**: For tracking dotfile configurations and allowing simple `git clone` deployment.
